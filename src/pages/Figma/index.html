Protótipo de Sistema de Gestão Escolar (SIGA)

Inspirado no SIGA, sistema de gerenciamento acadêmico usado pela FATEC-SP, o projeto teve como objetivo aprimorar a usabilidade e a experiência do usuário em uma interface de gestão educacional, pensada especificamente para dispositivos móveis. Esse foco em mobile visa atender às necessidades de alunos e professores, facilitando o acesso a informações acadêmicas de maneira intuitiva e prática, além de otimizar a navegação em telas menores. O projeto de prototipação do sistema de gestão escolar foi desenvolvido na disciplina de Interação Humano-Computador (IHC), ministrada pelo professor Felipe Maciel Rodrigues.

Tecnólogia Utilizada

Para o desenvolvimento das interfaces, utilizou-se o Figma, uma ferramenta amplamente utilizada para a criação de protótipos interativos de alta fidelidade. No Figma, é possível construir layouts complexos, testar interações, simular fluxos de navegação e compartilhar o projeto para revisão. Essas funcionalidades foram essenciais para criar um protótipo navegável que simula o funcionamento real do sistema, permitindo ajustes antes da implementação final. Com a aplicação dos conceitos de UI (Interface do Usuário) e UX (Experiência do Usuário), o Figma também facilitou a realização de testes de usabilidade, que foram fundamentais para otimizar a interface e garantir uma navegação prática.

Conclusão

Esse projeto representa uma aplicação prática dos conceitos de UI e UX aprendidos durante a disciplina de IHC, fortalecendo o portfólio acadêmico com um exemplo de design voltado para sistemas de gestão escolar. O desenvolvimento da prototipação ampliou a compreensão sobre o design de interfaces complexas, especialmente para plataformas móveis, além de reforçar a importância da usabilidade em contextos educacionais. Com isso, a solução proposta oferece uma visão prática e acessível, adaptada às necessidades do meio acadêmico e pronta para futuras implementações.
