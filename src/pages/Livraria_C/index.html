Sistema De Livraria em C

O projeto teve como objetivo desenvolver um sistema funcional que permite o cadastro, consulta, listagem e exclusão de dados relacionados a livros, autores, categorias e editoras. Utilizando a linguagem C, o sistema foi estruturado com o uso de structs e organizado de forma modular para facilitar a manutenção e evolução do código. Para garantir a persistência de dados entre execuções, foram utilizados arquivos binários com extensão .dat. O projeto foi desenvolvido como parte da disciplina de Estruturas de Dados, sob a orientação do professor Julio Fernando Lieira, durante o segundo bimestre, destacando-se pela aplicação prática de conceitos de programação e organização de dados.

Tecnólogias Utilizadas

O projeto foi desenvolvido totalmente em C, utilizando as bibliotecas padrão stdio.h e string.h para realizar operações de entrada e saída de dados e manipulação de strings. Para garantir a persistência das informações entre as 20 execuções do programa, os dados são armazenados em arquivos binários com a extensão .dat. Cada tipo de dado, como livros, clientes e vendas, é armazenado em arquivos separados, e a organização dos arquivos é feita por meio de constantes definidas com a diretiva #define, o que facilita a manutenção e a leitura do código, além de tornar o processo de gerenciamento de dados mais eficiente.

Funcionalidades

O sistema foi projetado utilizando structs, que são estruturas de dados fundamentais para organizar e agrupar informações relacionadas em um único bloco, facilitando o gerenciamento de dados. Cada struct é composta por campos que representam atributos específicos de uma entidade. Por exemplo, uma struct para livros pode incluir atributos como título, autor e estoque, enquanto uma para clientes pode armazenar nome, contato e endereço. Essa abordagem permite agrupar dados logicamente relacionados, reduzindo a complexidade do código e tornando-o mais legível e eficiente. Com as structs como base, o sistema implementa funcionalidades como cadastro, onde as informações são preenchidas e associadas a uma struct antes de serem salvas; consulta e listagem, que percorrem os dados armazenados para exibir ou buscar registros específicos; e operações de alteração e exclusão, que atualizam ou removem informações diretamente dentro das estruturas ou nos arquivos onde estão persistidas. Essa arquitetura proporciona flexibilidade, modularidade e facilidade na manipulação e persistência de dados.

Conclusão

Este projeto de livraria desenvolvido em C proporciona uma excelente oportunidade para compreender os conceitos de programação estruturada, como o uso de structs para organizar dados e a manipulação de arquivos binários para persistência. A implementação das operações de CRUD (criação, leitura, atualização e exclusão) permite uma gestão eficiente de livros, clientes, editoras e categorias, enquanto as funções de relatório ajudam a analisar o desempenho de vendas e a gestão de estoque. O projeto não só atende aos requisitos de um sistema básico de gestão de livraria, como também oferece uma base sólida para futuras melhorias, como a adição de novos recursos ou a integração com interfaces gráficas. Em suma, é uma ferramenta valiosa tanto para o desenvolvimento de habilidades em C quanto para a construção de soluções práticas e funcionais.
