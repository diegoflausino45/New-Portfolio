Circuito em Arduino

Desenvolvido na disciplina de Programação Orientada a Objetos Avançada, ministrada pelo professor Alciano Gustavo Genovez de Oliveira, o projeto do piano elétrico teve como objetivo criar um sistema interativo para reprodução de notas musicais utilizando a plataforma Arduino. Inicialmente, o circuito foi projetado e testado no Tinkercad, uma ferramenta que permitiu simular o funcionamento e validar o código antes de transferi-lo para o hardware real. Após essa etapa, o código foi implementado em um Arduino físico, onde ajustes finais foram realizados para garantir a funcionalidade em um ambiente real. O projeto buscou explorar a integração entre hardware e software, promovendo aprendizado prático e o desenvolvimento de habilidades em eletrônica e programação.

Componentes Utilizadas

O projeto do piano elétrico utilizou a plataforma Arduino Uno como controlador central para processar o código e gerenciar os sinais de entrada e saída. Um buzzer foi empregado para emitir frequências correspondentes às notas musicais, transformando sinais elétricos em ondas sonoras. As teclas foram representadas por botões conectados à placa do Arduino, cada um associado a uma frequência específica para gerar as notas desejadas. Essa combinação de componentes e a transição do ambiente virtual para o físico permitiram um desenvolvimento eficiente, proporcionando aprendizado prático em música e eletrônica.

Funcionalidades

O projeto do piano elétrico com Arduino implementa funcionalidades que permitem ao usuário reproduzir sons distintos através de botões conectados à placa. Cada botão é mapeado a um pino digital do Arduino e associado a uma frequência específica. O buzzer, conectado ao pino 13, emite os sons das notas musicais. No setup(), os pinos dos botões são configurados como entradas para detectar pressionamentos, enquanto o pino do buzzer é configurado como saída para emitir os sons. Durante o loop(), o estado de cada botão é lido com digitalRead() e armazenado em variáveis. Quando um botão é pressionado, a função tone() é chamada para gerar o som correspondente no buzzer, utilizando frequências que variam de 300 Hz a 1000 Hz, representando notas em tons crescentes. Um pequeno intervalo de 10 ms, inserido com delay(10), garante a suavidade na leitura dos botões e melhora a experiência de interação. Esse sistema permite criar um piano básico, onde cada tecla é representada por um botão, proporcionando uma experiência prática e interativa de aprendizado.

Conclusão

O projeto do piano elétrico com Arduino integrou eletrônica e música, utilizando componentes simples como o Arduino Uno, botões e um buzzer para reproduzir notas musicais de forma interativa. Além de reforçar o aprendizado técnico em eletrônica e programação, o projeto destacou o potencial criativo do Arduino, servindo como base para futuros aprimoramentos, como a expansão de notas e inclusão de efeitos sonoros.
