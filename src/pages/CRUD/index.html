CRUD (Create, Read, Update & Delete)

Desenvolvido na disciplina de Tópicos Especiais em Informática, sob orientação do professor Julio Fernando Lieira, o projeto consistiu na criação de um sistema CRUD para gestão de usuários e produtos, utilizando PHP, SQL e Bootstrap. A aplicação inclui funcionalidades de cadastro, listagem, edição, exclusão e controle de login, incorporando aspectos de segurança, como sanitização de dados, para prevenir vulnerabilidades. O projeto aplicou conceitos fundamentais do desenvolvimento web, com foco em organização do código e uma experiência de usuário fluida e segura, integrando a nota do primeiro bimestre.

Linguagens e Tecnologias Utilizadas

O projeto utiliza PHP, uma linguagem de programação de código aberto voltada para o desenvolvimento de sites dinâmicos e interativos, com integração direta ao HTML. Para gerenciar e manipular dados, utiliza SQL, que permite executar operações em bancos de dados relacionais, como consultas e atualizações. No front-end, foi utilizado o Bootstrap, um framework que facilita a criação de interfaces modernas e responsivas com componentes pré-desenvolvidos. O XAMPP foi escolhido como ambiente local de desenvolvimento, integrando servidor web e banco de dados para testes e implementação do sistema.

Funcionalidades

O projeto é um sistema de gestão de usuários com funcionalidades de cadastro, listagem, edição e exclusão. Utiliza PHP e SQL para processar e armazenar dados, com validação e sanitização para segurança. O cadastro verifica logins duplicados e exibe mensagens de feedback. A listagem apresenta dados em tabela com opções de edição e exclusão. A edição permite alterações por formulário, enquanto a exclusão conta com confirmação antes de processar a remoção. Tudo foi desenvolvido com boas práticas de organização, validação e feedback ao usuário.

Conclusão

O desenvolvimento do Sistema CRUD para usuários e produtos, permitiu a aplicação prática de conceitos fundamentais no desenvolvimento de sistemas web interativos e funcionais. Durante a construção do projeto, foram incorporados importantes aspectos de segurança, como a sanitização de dados, garantindo a proteção contra vulnerabilidades como injeção de código. Além disso, a estruturação do código seguiu boas práticas de organização e controle de fluxo, proporcionando uma experiência de usuário fluida e segura.
